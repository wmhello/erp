# 商品进销存管理系统

## 一、功能
商品信息及种类管理
> 进货管理
>
> 销售管理
>
> 打印统计报表
>
> 根据库存进行智能提醒进货
>
## 二、技术交流
> 微信1(xpyzwm） 微信2(common_program)

## 三、使用范围
1.作为库存系统基础模板使用

2.适用于毕业设计及相关产品演示
